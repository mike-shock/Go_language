# Go language: Surview

What makes Go a great language 
* simplicity: 25 keywords, all the language in your head
* compatibility promise
* interfaces: flexibility and a mechanism for painless evolution
* goroutines: structuring programs for concurrency
* fast speed of building and execution
* portability via cross-compiling
* rich standard library
* tools: automation on every step of development
